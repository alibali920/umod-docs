# OnHotAirBalloonToggled

## Usage

* Called right after a player has toggled a hot air balloon on or off
* No return behavior

## Examples

```csharp title=""
void OnHotAirBalloonToggled(HotAirBalloon balloon, BasePlayer player)
{
    Puts("OnHotAirBalloonToggled works!");
}
```
