# OnWaterPurified

## Usage

* Called after salt water has been converted to fresh water in a water purifier
* No return behavior

## Examples

```csharp title=""
void OnWaterPurified(WaterPurifier waterPurifier, float timeCooked)
{
    Puts("OnWaterPurified works!");
}
```
