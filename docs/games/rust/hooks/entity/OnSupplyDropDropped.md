# OnSupplyDropDropped

## Usage

* Called right after a cargo plane has dropped a supply drop
* No Return behavior

## Examples

```csharp title=""
void OnSupplyDropDropped(SupplyDrop supplyDrop, CargoPlane cargoPlane)
{
    Puts("OnSupplyDropDropped works!");
```
